## 5. Common Customer Scenarios

### Scenario 1: Small Business Going Digital
**Profile:** 25 employees, retail, basic IT
**Recommendation:** Microsoft 365 Business Standard ($312.50/month)
**Annual cost:** $3,750
**Template:** Use Microsoft 365 Collaboration template

### Scenario 2: Mid-Market Manufacturer
**Profile:** 150 employees, need ERP + productivity
**Recommendation:**
- M365 E3 for all (150 × $36 = $5,400/mo)
- Business Central for 10 users ($1,000/mo)
- Azure infrastructure ($2,000/mo)
**Total:** $8,400/month or $100,800/year
**Templates:** Azure Migration + Power Platform

### Scenario 3: Enterprise Digital Transformation
**Profile:** 1,000 employees, finance industry, high security needs
**Recommendation:**
- M365 E5 ($57,000/mo)
- Copilot for 300 users ($9,000/mo)
- Dynamics 365 Finance ($10,500/mo for 50 users)
- Azure compliance infrastructure ($25,000/mo)
**Total:** $101,500/month or $1,218,000/year
**Templates:** Security & Compliance + all others

### Scenario 4: Copilot-First Strategy
**Profile:** 200 knowledge workers, consulting firm
**Recommendation:**
- M365 E3 base license: 200 × $36 = $7,200/mo
- Copilot for all: 200 × $30 = $6,000/mo
**Total:** $13,200/month or $158,400/year
**ROI:** 10-20 hours saved per user/month
**Template:** Microsoft 365 Collaboration with Copilot focus

## 6. Competitive Positioning

### vs. Google Workspace
**Advantages:**
- Superior desktop apps (Excel, PowerPoint)
- Better enterprise security (Microsoft Defender)
- Integrated ERP/CRM (Dynamics 365)
- Azure cloud platform
- Copilot AI integration

**Pricing comparison:**
- Google Workspace Business Standard: $12/user/mo
- M365 Business Standard: $12.50/user/mo
- **Similar price, more features with Microsoft**

### vs. Salesforce
**Advantages:**
- Lower cost (D365 Sales at $65-105 vs Salesforce at $75-165)
- Integrated with Microsoft 365
- Better Office integration
- Power Platform included

**Pricing comparison:**
- Salesforce Sales Cloud Professional: $75/user/mo
- D365 Sales Professional: $65/user/mo
- **14% cost savings + M365 integration**

## 8. Sales Playbooks by Deal Size

### Small Deal ($5K-25K annual)
1. Focus on quick wins (M365 Business plans)
2. Self-service onboarding
3. Minimal customization
4. Use Business Basic/Standard templates

5. Use enterprise solution bundles

### Strategic Deal ($1M+ annual)

2. Digital transformation roadmap

## 9. Objection Handling

### "Too Expensive"
**Response:**
- Show ROI calculator (typically 200-400% ROI)
- Compare to cost of existing tools + labor
- Offer phased approach
- Mention volume discounts
**Reference:** Solution bundles pricing, ROI section

### "Too Complex"
**Response:**
- Microsoft FastTrack (free for large deployments)
- Partner support available
- Start with pilot program
- User training included
**Reference:** Copilot adoption strategies

### "Already Using Google/Competitor"
**Response:**
- Migration tools available (often free)
- Better desktop apps
- Superior security
- Azure ecosystem advantage
**Reference:** Competitive positioning section

### "Not Ready for Cloud"
**Response:**
- Hybrid options available
- Gradual migration path
- On-premises still supported
- Better disaster recovery
**Reference:** Azure migration template
